# hr_analytics-
Collection of interactive Power BI dashboards including Sales Analytics and HR Presence Insights.
## 2. Leap Presence Insights Dashboard

This HR analytics dashboard tracks employee presence, work-from-home trends, and sick leave patterns.

Key insights:
- Attendance percentage over time
- Work from home trends
- Sick leave patterns
- Employee-wise attendance tracking

Tools used:
- Power BI
- DAX
- Data Visualization
